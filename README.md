# dats
Compose music in a text editor.


Example:
```
NL 1 C3
NL 4 D3
...
```
# Installation
## Prerequisite
Bison and Flex

run "make" inside the src directory.

To test dats, execute `./dats < example.dats` or `./dats example.dats`.
Both would work anyhow.
